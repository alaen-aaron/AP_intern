# Tools Used

## Nmap
Used for network scanning and discovering open ports.

Example:
nmap 192.168.56.101

---

## Wireshark
Used to capture and analyze network packets.

---

## OpenSSL
Used for encryption and decryption.

Example:

openssl enc -aes-256-cbc -salt -in msg.txt -out encrypted.txt

openssl enc -aes-256-cbc -d -in encrypted.txt -out decrypted.txt
