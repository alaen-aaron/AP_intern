# Linux Command Cheat Sheet

pwd  
Shows current directory.

ls  
Lists files and directories.

cd  
Change directory.

mkdir  
Create a directory.

touch  
Create a file.

chmod  
Change file permissions.

chown  
Change file ownership.

apt update  
Update package list.

apt install  
Install software.

ifconfig / ip a  
Display IP address.

ping  
Test network connectivity.

netstat  
Display network connections.

traceroute  
Trace network path.
