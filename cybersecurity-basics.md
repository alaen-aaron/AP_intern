# Cybersecurity Basics

## CIA Triad

Confidentiality  
Protect sensitive information from unauthorized access.

Integrity  
Ensure data is not altered or tampered with.

Availability  
Systems and data should be accessible when needed.

---

## Common Cyber Attacks

Phishing  
Fraudulent attempts to steal credentials.

Malware  
Malicious software designed to harm systems.

DDoS  
Overloading a server with traffic.

SQL Injection  
Injecting malicious SQL queries into databases.

Brute Force  
Repeated password guessing.

Ransomware  
Encrypting files and demanding payment.
